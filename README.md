# Xpens Analytics

Xpens Analytics is a clean expense tracking dashboard for monitoring income, expenses, balance, and downloadable PDF reports.

## Features

- Simple landing page with product-style UI
- Session-based tracking that clears on refresh
- Income and expense analysis with charts
- PDF report export with summary and transactions
- Responsive layout for desktop and mobile
- Vercel-ready deployment config

## Getting Started

1. Install dependencies with `npm install`
2. Start the app with `npm start`
3. Open `http://localhost:3000`

## Build

Run `npm run build` to create the production build.

## Deploy

This project includes `vercel.json` and can be deployed directly to Vercel.
